# 🔐 Password Strength Analyzer

A cybersecurity-based Password Strength Analyzer built using Python and Streamlit.  
This application evaluates password security, provides suggestions for stronger passwords, generates secure passwords, and demonstrates SHA-256 hashing concepts.

---

# 📌 Features

✅ Password strength checking  
✅ Uppercase and lowercase validation  
✅ Number and special character detection  
✅ Real-time password feedback  
✅ Strong password suggestions  
✅ Secure password generator  
✅ SHA-256 password hashing  
✅ Streamlit web interface  
✅ Dark cybersecurity theme  

---

# 🚀 Technologies Used

- Python
- Streamlit
- Regex (`re`)
- Hashlib
- Random
- String module
